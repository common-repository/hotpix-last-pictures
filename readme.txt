== Description ==

A widget that shows the latest pictures of the site hotpix.org.
If you send visitors you will be posted on our TOPLIST in the left sidebar.
You'll get a return of visitors from around 150%

== Installation ==

1. Upload `hotpix.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
